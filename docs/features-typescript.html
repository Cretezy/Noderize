Feature: TypeScript

TypeScript support is built in to Noderize.

Simply set/add typescript to the languages option.

If you are not using JavaScript, you may want to remove the javascript languages for better build times.

See example.

Entry

When only using the typescript language option, the entry option is automatically set to src/index.ts.

If you are also using other languages but want your entry file to be a TypeScript file, simply change the entry option to src/index.ts.
